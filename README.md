# Automated Social Media Content Generation with n8n

An AI-powered automation workflow that summarizes news articles and automatically generates platform-specific posts for LinkedIn and X (Twitter).

## Tech Stack
- n8n
- Google Sheets
- Google Gemini (LLM)
- LinkedIn API
- X (Twitter) API

## How It Works
1. Detects new article links from Google Sheets
2. Summarizes the article using AI
3. Generates LinkedIn and X posts
4. Automatically publishes them

## Features
- Fully automated content pipeline
- Platform-optimized AI posts
- Saves time in content creation
